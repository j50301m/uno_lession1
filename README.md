# Arduino LED Control Project

*[中文](README_zh.md)*

This is a simple LED control project based on Arduino Uno, developed using PlatformIO development environment and Wokwi simulator.

## Project Description

This project implements a simple LED control system where the LED lights up when the button is pressed and turns off when the button is released.

## Development Environment

### Required Software

- [PlatformIO](https://platformio.org/) - For compiling and uploading code
- [Wokwi](https://wokwi.com/) - For circuit simulation (optional)

### Project Configuration

- **Platform**: Atmel AVR
- **Board**: Arduino Uno
- **Framework**: Arduino
